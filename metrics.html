Architectural Observations

The largest latency breakthrough came from eliminating oversized system prompts and moving visible-thought generation out of the synchronous inference path.

Earlier versions of the stack routinely exceeded 10k–13k prompt sizes, causing severe TTFT degradation and long conversational dead zones. Current runtime prompts generally operate between 1400–4300 characters depending on memory and retrieval injection.

Remaining latency spikes correlate strongly with:

• character_rag injection
• agentic memory retrieval
• long sliding-window orchestration blocks

Current Bottleneck

The system is no longer fundamentally model-bound. Current latency overhead is dominated by orchestration and contextual augmentation rather than raw inference throughput.
